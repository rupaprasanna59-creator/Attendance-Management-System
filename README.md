# Attendance-Management-System
Attendance Management System using Python and Tkinte
This is a simple Attendance Management System developed using Python and Tkinter GUI.
The application allows users to:
Add student names
Mark students as Present
Mark students as Absent
View attendance records
Search student attendance status
Save attendance records into a text file
Technologies Used
Python
Tkinter GUI
Text File Handling
Features
Add Student
Mark Present
Mark Absent
Save Attendance
Search Student Record
How to Run
Install Python.
Download the project files.
Open the terminal or command prompt.
Run:

python attendance.py

Output

The application opens a graphical user interface where attendance can be managed easily.
